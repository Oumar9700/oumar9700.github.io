CartoDataSup · Analyse territoriale Parcoursup

Power BI React Django

Les données Parcoursup sont publiques, mais personne ne les lisait vraiment à l'échelle locale. On a construit un outil pour changer ça.

Contexte

Mission confiée par Kaleino : exploiter les données open data du Ministère de l'Enseignement Supérieur pour produire des indicateurs territoriaux utiles aux décideurs locaux. ~9 000 lignes de données à traiter.

Résultat

Application full-stack (Django + React.js + Chart.js) avec 11 indicateurs clés : taux d'admission, pression sur les formations, mixité, statut boursier, disparités territoriales. Dashboards Power BI conçus en parallèle comme base de référence visuelle.
